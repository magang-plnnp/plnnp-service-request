# PLN NP Service Request System

Sistem permintaan layanan internal untuk **PLN NP** yang mencakup:

-   Permintaan makanan
-   Permintaan kendaraan dinas

Website ini terdiri dari:

-   Halaman landing page untuk pengguna mengajukan permintaan
-   Dashboard admin untuk memantau dan mengelola data permintaan
